# social-media-feed-aggregator

A simple web-based project that displays content from multiple social media platforms in one place. This interface showcases feeds from platforms like YouTube and TikTok in a clean and organized layout.

## 🚀 Features

* Displays social media content in card format
* Supports multiple platforms (YouTube, TikTok)
* Clean and responsive UI design
* Simple and easy to use
